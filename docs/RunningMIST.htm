3. Running MIST

MIST is a 32-bit Windows application. To run MIST, find the folder containing the executable program and double-click on its icon. You can create a shortcut to the program by right-clicking on the icon, then choosing “Create Shortcut.” The shortcut can then be moved to the Start Menu or the Desktop.
