2. Installing MIST

The latest released version of MIST can be obtained from the MIST GitHub Latest Release page. Download the file winMIST.exe and any example MIST programs (*.mist). It is suggested that users create a directory and copy all these files into that directory.

The release version of the source code can also be downloaded from this page. Programmers can also obtain the development version of the source code from the MIST GitHub site.
